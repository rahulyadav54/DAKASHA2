# **App Name**: SmartRead AI Tutor

## Core Features:

- Content Input & Extraction: Allows users to upload text passages, PDF documents, or articles, from which the system automatically extracts the text content for processing.
- AI Question Generation: Utilizes AI to automatically generate contextually relevant Multiple Choice, Short Answer, True/False, and Fill-in-the-blanks questions from uploaded content.
- Semantic Answer Evaluation: Compares student answers written in their own words against reference answers using semantic similarity with an AI tool, providing a correctness score, detailed explanation feedback, and suggested improvements.
- Adaptive Learning Engine: Automatically adjusts the difficulty level of quizzes based on student performance (beginner, intermediate, advanced) and uses an AI tool to detect the reading complexity of new content.
- User Management & Access Control: Implements distinct user roles (Student, Teacher, Parent, Admin) with secure JWT authentication and role-based access control.
- Personalized Dashboards & Analytics: Provides dedicated dashboards for students, parents, and teachers showcasing quiz score history, reading improvement, weak topic areas, and class performance using charts and graphs.
- Interactive Quiz Interface: A user-friendly interface for students to take quizzes generated from uploaded content and view their results, feedback, and progress.

## Style Guidelines:

- Primary color: A vibrant, clear blue (#2680D9) for an energetic and trustworthy educational feel.
- Background color: A very light, desaturated blue-grey (#F0F2F4) to ensure excellent readability and a calm learning environment.
- Accent color: A rich, deep violet (#422D66) used sparingly for highlights and important calls-to-action, providing sophisticated contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly tech-forward and distinctive appeal. Body text font: 'Inter' (sans-serif) for clean, highly readable, and objective presentation of information.
- Clean, modern line icons that clearly convey educational concepts and system actions, ensuring intuitive navigation.
- A structured and responsive grid-based layout prioritizing content clarity and hierarchy across all screen sizes, with ample white space to prevent visual clutter.
- Subtle and purposeful animations and transitions to provide feedback on user interactions and smoothly guide attention, enhancing the user experience without distraction.
