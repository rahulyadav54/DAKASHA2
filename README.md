# SmartRead AI Tutor 🚀

SmartRead AI is an advanced reading comprehension and adaptive learning platform built for the AI Hackathon. It transforms static text and PDF documents into interactive, personalized learning experiences.

## ✨ Features

- **AI Question Generation**: Automatically creates MCQs, Short Answer, True/False, and Fill-in-the-blank questions from any text.
- **Semantic Answer Evaluation**: Uses Gemini AI to understand the meaning of student answers, providing nuanced feedback beyond simple keyword matching.
- **AI Study Tutor**: A persistent conversational assistant that helps explain complex concepts from the reading material.
- **Adaptive Learning**: Analyzes performance to suggest reading levels and adjust difficulty.
- **Study Tools**: Generates comprehensive Study Guides and Flashcard sets automatically.
- **Parent/Teacher Portal**: Detailed analytics and progress tracking.
- **Gamification**: Achievement system to motivate learners.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **AI Engine**: Google Gemini (via Genkit)
- **Database & Auth**: Firebase (Firestore & Firebase Auth)
- **Styling**: Tailwind CSS & Shadcn UI
- **Language**: TypeScript

## 🚀 Getting Started

To get this project running locally or to push it to your own repository:

### GitHub Setup
If you want to push this code to your repository, run these commands in your terminal:

```bash
git init
git add .
git commit -m "Initial commit: SmartRead AI Hackathon Project"
git branch -M main
git remote add origin https://github.com/rahulyadav54/Ai_hackathon.git
git push -u origin main
```

### Environment Variables
Ensure you have your Gemini API Key set in a `.env` file:
```env
GOOGLE_GENAI_API_KEY=your_api_key_here
```
